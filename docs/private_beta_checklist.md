# Private Beta Checklist

## Access

- [ ] invite list defined
- [ ] operator/admin access confirmed
- [ ] beta users know product is post-hoc only

## Core flow

- [ ] signup works
- [ ] login works
- [ ] upload works
- [ ] duplicate skip works
- [ ] Today renders
- [ ] Review renders
- [ ] Brain renders

## Product safety

- [ ] no live assistance language
- [ ] no operator-only internals leak publicly
- [ ] zero-hand failures do not fabricate interpretation
- [ ] user data isolation is confirmed

## Billing

- [ ] plan labels are visible
- [ ] account page explains plan state
- [ ] free vs paid boundary is understandable

## Support

- [ ] support response template exists
- [ ] upload failure path is documented
- [ ] interpretation escalation path is documented

## Launch decision

- [ ] ready for private beta
- [ ] not ready, hold launch
